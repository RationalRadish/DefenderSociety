Tool description:

  • User-Agent is mostly used to set request headers in the crawler program, which is the simplest disguise for crawler programs
  • This tool automatically generates User-Agent for different platforms, different systems and different browsers by setting different parameters
  • When the parameter is not selected, it means that no restriction is set, the User-Agent is randomly generated, and the User-Agent corresponding to the parameter is generated when the parameter is set
  • When there is a conflict between the set parameters, a suitable User-Agent will not be generated, and an error message will be displayed
